# Tupll Area Report — Austin Showroom Site Comparison (prototype)

A drive-time **trade-area** comparison of three candidate outdoor-living showroom sites in Greater Austin (West Campus, Cedar Park, Bee Cave), built as a Tupll free-report prototype.

**Live site:** https://ambientarray.github.io/tupll-area-report/

Real data: OpenRouteService 20-minute drive-time isochrones, ACS 2024 tract-aggregated demographics, Google Places (via Apify) competitor landscape, Leaflet + CARTO basemap. This is a prototype for critique — trajectory direction is a structural read pending time-series wiring.
